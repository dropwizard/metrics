######################################################################################################
Metrics is a Java library which gives you unparalleled insight into what your code does in production.
######################################################################################################

**Metrics** provides a powerful toolkit of ways to measure the behavior of critical components
**in your production environment.**

With modules for common libraries like **Jetty**, **Logback**, **Log4j2**, **Apache HttpClient**,
**Ehcache**, **JDBI**, **Jersey** and reporting backends like **Graphite** and **InfluxDB**, Metrics
provides you with full-stack visibility.
